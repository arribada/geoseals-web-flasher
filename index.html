GeoSeals Hub Firmware Installer

This is the GeoSeals Hub Firmware Installation tool. For more information about the GeoSeals project, read the blogpost.

1. Connect the GeoSeals Hub via USB cable to this machine.

2. Then select your development branch:

3. Click the 'connect' and select the CP2104 USB to UART Bridge. Then install your chosen firmware branch.

Unfortunately your web browser doesn't support flashing GeoSeals Hub hardware! Please try a newer version of Google Chrome or Microsoft Edge.

Please ensure to verify firmware version (e.g. v1.0.0) before clicking install.

View device logs

To view the device logs follow instructions 1-3 but instead select 'Logs & Console'.
